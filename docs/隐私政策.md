# 课时小伙伴隐私政策

更新日期：2026-06-23

本政策适用于微信小程序「课时小伙伴」。本小程序用于帮助机构、老师和家长进行课时管理，包括课程管理、学员管理、课时充值、上课打卡、记录查询、成员绑定和意见反馈。

发布前请将本文件中的主体名称、联系方式等信息替换为实际运营主体信息。

## 我们收集的信息

为实现课时管理功能，本小程序可能收集和处理以下信息：

1. 微信身份标识：通过微信小程序云开发能力获取用户的 OpenID，用于识别当前登录用户和绑定角色。
2. 机构和成员信息：机构名称、成员角色、成员昵称、绑定码、成员绑定状态。
3. 课程信息：课程名称、默认扣课时数、课程备注、课程启用状态。
4. 学员信息：学员姓名、关联课程、监护人姓名、联系方式、剩余课时数、备注、启用状态。
5. 课时记录：充值记录、打卡记录、补打卡记录、撤销打卡记录、撤销充值记录，以及对应的课时数、目标日期、操作日期和操作人。
6. 操作日志：为避免误删误改后无法追溯，系统会记录部分管理操作、操作时间、操作人和关联数据。
7. 意见反馈：用户主动提交的反馈内容和联系方式。

## 我们不收集的信息

当前版本不主动收集以下信息：

- 精确地理位置。
- 通讯录。
- 相册、摄像头、麦克风内容。
- 支付银行卡信息。
- 身份证件信息。
- 与本小程序功能无关的浏览行为。

## 信息使用目的

我们使用上述信息用于：

- 识别用户所属机构和角色。
- 展示课程、学员、剩余课时和课时记录。
- 支持管理员管理课程、学员和成员。
- 支持老师进行打卡、补打卡和撤销打卡。
- 支持家长查看已绑定孩子的课时余额和记录。
- 处理用户反馈和问题排查。
- 保留必要操作日志，降低误删、误改造成的数据风险。

## 信息访问控制

本小程序按角色限制数据访问：

- 管理员：可管理本机构课程、学员、成员、充值、打卡、撤销、记录和操作日志。
- 老师：可查看学员，进行打卡、补打卡、撤销打卡，并查看记录。
- 家长：只读访问，仅可查看已绑定学员的剩余课时、打卡记录和充值记录。

不同机构之间的数据通过机构 ID 隔离。

## 信息共享

除以下情况外，我们不会主动向第三方共享用户数据：

- 按法律法规要求提供。
- 为实现微信小程序基础运行能力，由微信小程序和云开发平台提供必要的基础服务。
- 用户或机构管理员明确要求导出、迁移或删除数据。

当前版本不接入广告平台，不向第三方出售用户数据。

## 信息删除

用户可以通过小程序「我的」页面中的「意见反馈」提交数据删除请求，并说明需要删除的数据范围和联系方式。

机构管理员可以在小程序内删除课程、学员、成员绑定等数据。为便于追溯，课时记录和操作日志可能会保留一段时间；如需彻底删除机构数据，需要由机构负责人提出请求后统一处理。

详细流程见 `docs/数据删除说明.md`。

## 未成年人信息

本小程序可能包含学员姓名、课程和课时记录。机构管理员在录入学员信息前，应确保已经获得必要授权，并避免录入与课时管理无关的敏感个人信息。
