# 云开发部署说明

本文档说明如何把当前 MVP 跑成「云开发后端版」。

## 1. 开通云开发

云开发不能使用「测试号」。必须使用微信公众平台注册的小程序 AppID。

如果微信开发者工具报错：

```json
"reason": "测试号不能使用云服务"
```

说明当前项目仍然是测试号模式。需要先去微信公众平台注册正式小程序，拿到 AppID 后重新导入或修改项目 AppID。

1. 打开微信开发者工具。
2. 导入项目：`C:\deng\keshiji`。
3. 确认项目 AppID 不是测试号。
4. 顶部点击「云开发」。
5. 按提示开通云开发环境。
6. 记下环境 ID，例如：`prod-xxxx`。

正式 AppID 获取入口：

```text
https://mp.weixin.qq.com/
```

如果你只创建了一个云环境，项目可以先不填写环境 ID，微信开发者工具会使用当前默认环境。若有多个环境，建议填写：

```js
// utils/config.js
module.exports = {
  cloudEnv: "你的云环境ID"
};
```

## 2. 部署云函数

当前后端云函数目录：

```text
cloudfunctions/api
```

在微信开发者工具里：

1. 找到左侧 `cloudfunctions/api`。
2. 右键 `api`。
3. 选择「上传并部署：云端安装依赖」。
4. 等待部署完成。

云函数会自动尝试创建以下数据库集合：

```text
organizations
users
memberships
courses
students
lesson_records
```

如果自动创建失败，可以在云开发控制台的「数据库」里手动新建这些集合。

## 3. 数据库权限

MVP 阶段建议数据库集合权限设置为：

```text
仅创建者及管理员可读写
```

实际业务读写都走云函数 `api`，前端不直接操作数据库。正式发布前应继续保持「云函数校验权限，数据库不直接开放给前端」。

## 4. 首次运行

部署完成后点击「编译」。

首次进入小程序时，云函数会根据当前微信用户自动创建：

- 一个用户 `users`
- 一个「课时小伙伴演示机构」`organizations`
- 一条管理员成员关系 `memberships`
- 三个默认课程 `courses`
- 两个演示学员 `students`
- 一条演示课时流水 `lesson_records`

然后你可以继续测试：

- 首页统计
- 新增学员
- 充值课时
- 上课打卡消课
- 查看记录
- 重置演示数据

## 5. 常见问题

### 5.1 提示「云函数调用失败」

检查：

- 是否已经开通云开发。
- `cloudfunctions/api` 是否已经上传并部署。
- `utils/config.js` 里的 `cloudEnv` 是否填错。
- 微信开发者工具右上角云环境是否选对。

### 5.2 提示「please select an env in the editor (cloudfunctionRoot)」

这个错误表示微信开发者工具还没有在当前项目里选中云开发环境。处理步骤：

1. 确认当前项目 AppID 是正式小程序 AppID，不是测试号。
2. 在微信开发者工具顶部点击「云开发」。
3. 进入云开发面板后，确认当前环境选择为：

```text
cloud1-d8gtkgh0n2f7c8c5f
```

4. 回到代码编辑器。
5. 如果左侧 `cloudfunctions` 目录没有云朵图标，关闭项目后重新导入。
6. 重新右键 `cloudfunctions/api`，选择「上传并部署：云端安装依赖」。

注意：只在 CloudBase Console 创建环境还不够，微信开发者工具本身也必须选中这个环境。

### 5.3 提示集合不存在

进入「云开发」->「数据库」，手动创建：

```text
organizations
users
memberships
courses
students
lesson_records
```

然后重新编译。

### 5.4 页面一直没有数据

打开微信开发者工具 Console，看是否有云函数报错。常见原因是：

- 云函数未部署。
- 云环境 ID 错误。
- 数据库权限过严。
- 当前 AppID 没有开通云开发。

## 6. 当前云函数接口

前端统一调用云函数：

```text
api
```

通过 `action` 区分接口：

| action | 功能 |
| --- | --- |
| login | 登录并创建用户/机构 |
| dashboard | 首页统计和最近记录 |
| studentList | 学员和课程列表 |
| studentCreate | 新增学员 |
| studentRecharge | 充值课时 |
| lessonCheckin | 上课打卡消课 |
| recordList | 课时流水 |
| profileGet | 我的页面资料 |
| resetDemoData | 重置演示数据 |
