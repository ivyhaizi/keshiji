# 课时小伙伴用户协议

更新日期：2026-06-23

欢迎使用微信小程序「课时小伙伴」。本协议用于说明本小程序的使用规则、用户责任和服务边界。发布前请将主体信息和联系方式替换为实际运营主体信息。

## 服务内容

课时小伙伴是一款面向机构、老师和家长的课时管理工具，当前主要提供以下功能：

- 课程管理。
- 学员管理。
- 课时充值。
- 当日打卡和补打卡。
- 撤销打卡和撤销充值。
- 课时记录查询。
- 操作日志查看。
- 成员绑定和角色管理。
- 家长只读查看孩子课时余额和记录。
- 意见反馈。

## 用户角色

本小程序提供三类角色：

- 管理员：负责机构数据管理，包括课程、学员、充值、打卡、撤销、记录、操作日志和成员管理。
- 老师：负责查看学员、打卡、补打卡、撤销打卡和查看记录。
- 家长：只能查看已绑定学员的剩余课时、打卡记录和充值记录，不能修改数据。

## 使用要求

用户使用本小程序时应遵守以下要求：

- 不录入违法、虚假、侵权或与课时管理无关的信息。
- 不冒用他人身份或绑定码。
- 不恶意删除、篡改、刷写机构数据。
- 妥善保管绑定码，避免无关人员加入机构。
- 机构管理员应确保录入学员和家长信息前已经获得必要授权。

## 数据准确性

本小程序提供课时记录工具，具体课程安排、课时扣减规则、充值约定和纠纷处理由机构与学员或家长自行确认。

管理员和老师应在操作前核对学员、课程、课时数和日期。对于误操作，可使用撤销打卡、撤销充值等功能进行修正；操作日志会保留必要追溯信息。

本小程序依赖微信小程序和微信云开发服务。因网络、微信平台、云开发服务、设备兼容性或维护升级导致的短时不可用，属于当前版本的服务边界。
