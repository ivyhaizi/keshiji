# 课时小伙伴品牌注册信息

## 小程序注册名称

建议注册名称：

```text
课时小伙伴
```

名称含义：

- 「课时」直接表达产品核心场景：课时管理、充值、消课、记录。
- 「小伙伴」比「管理系统」更轻量，适合免费试用工具，也降低家长和老师的使用距离感。

## 小程序简介

可用于微信公众平台简介：

```text
面向教培机构和老师的课时管理工具，支持学员管理、课时充值、上课打卡消课和课时记录查询。
```

## 图标设计说明

图标采用蓝绿渐变底色，中心是「时钟 + 双人伙伴」图形：

- 时钟代表课时、打卡、记录。
- 双人圆点代表老师、学员、家长之间的协同。
- 不放文字，避免小尺寸下文字不可读。
