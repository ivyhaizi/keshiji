# 课时小伙伴 MVP 小程序

这是一个原生微信小程序 MVP，用于免费试用版的课时管理场景。注册名称建议使用「课时小伙伴」。当前版本使用微信云开发保存数据，适合产品验证、演示和早期试用；后续可继续扩展角色权限、家长端和消息通知。

## 已实现功能

- 首页经营概览：学员数、剩余课时、最近记录
- 课程管理：新增课程、查看课程列表、编辑课程、删除未使用课程
- 学员管理：按「学员-课程」管理报名项、查看课时余额、快速充值、编辑、删除
- 上课记录：按最近记录查看消课流水
- 我的/设置：数据导出预览、清空本地演示数据

## 参考功能范围

参考「课时记」小程序的主要方向，本 MVP 聚焦：

- 老师、班级、学员等教务管理
- 课时余额和课时记录
- 消课、批量/补卡打卡作为后续扩展能力
- 家校协同、课时消耗通知、备份导出、多端同步的产品预留

## 运行方式

1. 打开微信开发者工具。
2. 选择「导入项目」。
3. 项目目录选择本仓库：`C:\deng\keshiji`。
4. AppID 可先选择测试号，或填入正式小程序 AppID。
5. 开通云开发，并按 [docs/云开发部署.md](docs/云开发部署.md) 部署云函数。
6. 编译运行。

## 发布准备

正式上线免费试用前，建议补齐：

- 登录与机构空间：微信登录、机构创建、成员角色
- 家长端绑定：家长查看课表、余额和上课记录
- 消息通知：上课提醒、消课通知、余额不足提醒
- 数据备份：CSV/Excel 导出和云端自动备份
- 隐私合规：隐私协议、用户授权、数据删除入口
